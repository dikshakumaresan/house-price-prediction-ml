**House Price Prediction using Machine Learning**

**📌 Project Overview**

This project focuses on predicting house prices using Machine Learning regression algorithms. The workflow includes data preprocessing, feature scaling, exploratory data analysis, model training, and performance evaluation. Multiple regression models are implemented and compared to identify the most accurate approach for house price prediction.

**🚀 Features**

Data preprocessing and cleaning
Feature scaling using StandardScaler
Train-test data splitting
Multiple regression models implementation
Model performance comparison
Evaluation using MAE, MSE, and R² Score
House price prediction based on input features

**🛠️ Technologies Used**

Python
Pandas
NumPy
Matplotlib
Seaborn
Scikit-learn
Jupyter Notebook / VS Code

**📂 Dataset**

The project uses a housing dataset containing various features related to houses, such as:
Median Income
House Age
Average Rooms
Average Bedrooms
Population
Households
Latitude
Longitude

Target Variable:
House Price

**📊 Machine Learning Models**

The following regression models are implemented and evaluated:
Linear Regression
K-Nearest Neighbors (KNN) Regressor
Decision Tree Regressor
Random Forest Regressor

**⚙️ Project Workflow**

1. Data Preprocessing
Load dataset
Handle missing values
Feature selection
Data scaling using StandardScaler
2. Data Splitting
Training Set
Testing Set
3. Model Training
Train multiple regression algorithms on the processed dataset.
4. Model Evaluation
Performance is measured using:
Mean Absolute Error (MAE)
Mean Squared Error (MSE)
R² Score
5. Prediction

Predict house prices using the trained model.

**📈 Results**

The performance of different regression models is compared to determine the best-performing model for house price prediction.
